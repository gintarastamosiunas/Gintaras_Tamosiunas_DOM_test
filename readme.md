# CA DOM atsiskaitymas

## Reikiamos programos
- node.js
- narsykle

## Instaliavimas
- sio failo direktorijoje ivykdykite komanda: 
- npm install

## Projekto paleidimas
1. Sio failo direktorijoje paleiskite serveri:
* windows - npm run start-win
* unix/Mac - npm run start-unix
2. Narsykleje atidarykite faila index.html
